#### 位置付け

#### 目標
- まずはGoFのデザインパターンを習得することを目標とする

#### 進め方
- 毎週何かしら（ちょこっとでもOKなので）進めていく
- 課題を解く。ベースでやる(やりたい)
- GoFの本には、Zipでも範囲解答があるので一旦自分で書いてみて、照らし合わせてみる

#### その他
- リポジトリはGoF以外のCodeを載せてもOKとする（PythonでもSQLでもOK）
- 何かモノを作ってみたい(2022-10-31メモ)
